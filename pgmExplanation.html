The DO/WHILE Loop Explained

Java Do/While Loop

Loops can execute a block of code as long as a specified condition is reached. Loops are handy because they save time, reduce errors, and they make code more readable. [1]

The do/while loop is a variant of the while loop. This loop will execute the code block once, before checking if the condition is true, then it will repeat the loop as long as the condition is true.

Sample Code

            class LoopExample {
            public static void main(String args[]) {
            int counter = 1;
            do {
            System.out.println("Hello World");
            counter++;
                } while( counter <= 5 );
              }
            }

Output

              Hello World
              Hello World
              Hello World
              Hello World
              Hello World

Explanation

For this example, we used an integer counter variable, initially set to 1. As for the condition, we used counter <= 5. Do not forget to increase the variable used in the condition, otherwise the loop will never end! [2]

In the loop body, we printed the string "Hello World" using the println() method.

We also used the ++ increment operator to increment the counter variable value by 1.
